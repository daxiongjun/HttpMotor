# HttpMotor
由若干子模块组成的基于业务开发中的底层支持库，（安全处理，接口生成器，日志服务，消息推送，常规监控，异常处理 and so on）,目前处于开发阶段。
=================================================================================================================================
I,异常处理模块：解决的问题是当出现网络请求错误时，可以明确的区分客户端/服务端发生错误的原因，用户可以定义自己关注的错误码，定义一个唯一的名称，选择你想要错误状态码，这个设置会格外地对这两种网络错误进行额外的处理，比如，控制台错误输出，日志文件写入。\n
